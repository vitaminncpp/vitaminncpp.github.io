Detailed Description

The Khazad legacy-level block cipher.

References

The Khazad algorithm was developed by Paulo S. L. M. Barreto and Vincent Rijmen.

See P.S.L.M. Barreto, V. Rijmen, `‘The Khazad legacy-level block cipher,’' NESSIE submission, 2000.

Author
Paulo S.L.M. Barreto
Vincent Rijmen.
Version
2.0 (2001.09.24)

Differences from version 1.0:

  • Original S-box replaced by the tweaked, hardware-efficient version.
