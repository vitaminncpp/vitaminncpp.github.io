Detailed Description

The Anubis block cipher.

References

The Anubis algorithm was developed by Paulo S. L. M. Barreto and Vincent Rijmen.

See P.S.L.M. Barreto, V. Rijmen, `‘The Anubis block cipher,’' NESSIE submission, 2000.
